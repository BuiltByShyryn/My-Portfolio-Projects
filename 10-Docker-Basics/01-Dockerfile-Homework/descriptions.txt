Dockerfile: это файл, где я пишу команды, чтобы Docker сделал образ.

FROM: указывает базовый образ, на основе которого создаётся новый образ.
Я говорю Docker: “возьми Python или Ubuntu и работай с ним”.

WORKDIR: это папка внутри контейнера, где я работаю.
Типа: “перейди сюда и делай всё здесь”.

COPY: копирует файлы с локального компьютера внутрь контейнера.

CMD: задаёт команду, которая выполняется при запуске контейнера.

Layer: это слой образа. Каждая инструкция в Dockerfile создаёт отдельный слой.

Build: это когда Docker собирает образ из Dockerfile.

Build Context: это папка, которую Docker видит и может использовать при сборке.
